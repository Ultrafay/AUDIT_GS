# APIs and Libraries Used in Invoice OCR Project

## External Libraries (Third-Party)

### 1. OpenCV (cv2)
- **Purpose**: Image processing and computer vision
- **Used in**: preproces.py, run_ocr.py, tables.py, handler.py, manual_extracter.py, converter.py
- **Key APIs**:
  - cv2.imread() - Read images
  - cv2.imwrite() - Write images
  - cv2.threshold() - Image thresholding
  - cv2.bitwise_not() - Image inversion
  - cv2.erode() - Morphological erosion
  - cv2.dilate() - Morphological dilation
  - cv2.getStructuringElement() - Create structuring elements
  - cv2.line() - Draw lines
  - cv2.rectangle() - Draw rectangles
  - cv2.resize() - Resize images
  - cv2.addWeighted() - Blend images
  - cv2.namedWindow(), cv2.imshow(), cv2.waitKey(), cv2.destroyWindow() - GUI display

### 2. Pytesseract
- **Purpose**: OCR (Optical Character Recognition) via Tesseract
- **Used in**: run_ocr.py, tables.py, manual_extracter.py
- **Key APIs**:
  - pytesseract.image_to_string() - Extract text from images
  - pytesseract.tesseract_cmd - Set Tesseract executable path

### 3. pdf2image
- **Purpose**: Convert PDF pages to images
- **Used in**: converter.py
- **Key APIs**:
  - convert_from_path() - Convert PDF to list of PIL images

### 4. NumPy (numpy)
- **Purpose**: Numerical operations and array handling
- **Used in**: main.py, preproces.py, run_ocr.py, tables.py, extraction.py, manual_extracter.py
- **Key APIs**:
  - np.array() - Create arrays
  - np.empty() - Create empty arrays
  - np.copy() - Copy arrays

### 5. Pillow (PIL)
- **Purpose**: Image handling
- **Used in**: run_ocr.py, tables.py, manual_extracter.py
- **Key APIs**:
  - Image module (imported but used alongside pdf2image)

### 6. XlsxWriter
- **Purpose**: Create Excel (.xlsx) files
- **Used in**: main.py
- **Key APIs**:
  - xlsxwriter.Workbook() - Create workbook
  - worksheet.write() - Write to cells
  - workbook.add_worksheet() - Add worksheets
  - workbook.close() - Save and close

## Python Standard Library

### 1. tkinter
- **Purpose**: Desktop GUI
- **Used in**: main.py
- **Key APIs**:
  - Tk() - Main window
  - Frame, LabelFrame - Container widgets
  - Label, Entry, Button - Input/display widgets
  - root.mainloop() - Event loop

### 2. pathlib
- **Purpose**: File path manipulation
- **Used in**: All modules
- **Key APIs**:
  - Path.cwd() - Current working directory
  - Path.joinpath() - Join paths
  - Path.mkdir() - Create directories

### 3. os
- **Purpose**: Operating system interface
- **Used in**: main.py
- **Key APIs**:
  - os module (general file operations)

### 4. subprocess
- **Purpose**: Run external commands
- **Used in**: run_ocr.py (fallback for Tesseract)
- **Key APIs**:
  - subprocess.check_output() - Run command and capture output

### 5. io
- **Purpose**: I/O operations
- **Used in**: extraction.py
- **Key APIs**:
  - io module (stream handling)

## Internal Modules (Project-Specific)

- constants.py - Configuration (tesseract_path, filename, flags)
- converter.py - PDF/image conversion
- preproces.py - Image preprocessing
- run_ocr.py - OCR execution
- extraction.py - Text field extraction
- tables.py - Table data extraction
- handler.py - Manual coordinate selection
- manual_extracter.py - Manual field extraction
