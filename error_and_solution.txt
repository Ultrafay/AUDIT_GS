ERROR REPORT AND SOLUTION
=========================

Issue: Application crashing during startup.

Error Message
-------------
pdf2image.exceptions.PDFInfoNotInstalledError: Unable to get page count. Is poppler installed and in PATH?

Explanation
-----------
The application tries to convert the input PDF (Sample8.pdf) into images using the `pdf2image` library. 
`pdf2image` is just a Python wrapper; it requires a separate software tool called **Poppler** to be installed on your computer to actually convert the PDF files. 
Since Poppler is not installed (or not configured in your system PATH), the application fails.

Solution Steps
--------------
You need to manually install Poppler on your Windows machine.

1.  **Download Poppler**:
    Go to the latest release page (e.g., https://github.com/oschwartz10612/poppler-windows/releases/) and download the zip file (e.g., `Release-24.02.0-0.zip`).

2.  **Extract Files**:
    Extract the downloaded zip file to a permanent location on your computer. 
    Recommended: `C:\Program Files\poppler`

3.  **Update System Environment Variables (PATH)**:
    - Press the **Windows Key** and search for "Environment Variables".
    - Click on **"Edit the system environment variables"**.
    - Click the **"Environment Variables..."** button.
    - Under **"System variables"** (bottom section), look for the variable named **Path** and select it.
    - Click **Permissions** or **Edit**.
    - Click **New** and paste the path to the `bin` folder inside your extracted poppler folder.
      Example: `C:\Program Files\poppler\poppler-24.02.0\Library\bin`
    - Click **OK** on all dialogs to save.

4.  **Restart**:
    You MUST restart your terminal (PowerShell/Command Prompt) or VS Code for the changes to take effect.

5.  **Verify**:
    Open a new terminal and run: `pdftoppm -h`
    If it prints usage information, Poppler is correctly installed.

6.  **Run the Project**:
    Run `python main.py` again.
